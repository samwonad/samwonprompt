# signboard_patterns.md

# Purpose

Reusable signboard concept patterns for image2 prompt workflows.

Use these patterns to help designers explore storefront and facade concepts faster.

# Signboard Priorities

Prioritize:

- instant business recognition
- strong name visibility
- distance readability
- facade compatibility
- durable commercial tone

Avoid:

- poster-like layouts
- excessive detail
- text that only works up close
- final fabrication instructions

# Pattern 1: Name-First Storefront

Use when the business name must dominate.

```text
Composition:
Make the business name the strongest visual element.
Use a simple horizontal signboard structure with clear margins.
Keep secondary information minimal.
Optimize for fast recognition from street distance.
```

# Pattern 2: Category + Name

Use when customers need to understand what the business sells immediately.

```text
Composition:
Balance the business category and name.
Place the category cue near the name without competing with it.
Use simple icon or graphic support only if it improves recognition.
Keep the sign readable from moving pedestrian or car traffic.
```

# Pattern 3: High-Contrast Retail Promo

Use for short-term promotional signage.

```text
Composition:
Prioritize the promotion message and discount.
Use a bold hierarchy with large offer text and minimal supporting copy.
Keep contrast high and visual elements simple.
Make the promotion readable from a distance.
```

# Pattern 4: Premium Minimal Facade

Use for refined stores, clinics, studios, and service brands.

```text
Composition:
Use restrained spacing and a clean name-first hierarchy.
Limit colors and graphic elements.
Prioritize trust, clarity, and facade compatibility.
Avoid visual clutter or aggressive promotional styling.
```

# Prompt Skeleton

```text
Subject:
Create a concept direction for a [store/business type] signboard.
Business name: [name].
Category or service: [category].
Required text: [required copy].

Style:
Use a [tone] signboard style with [typography direction].
Color direction: [palette or contrast].
The design should feel appropriate for [street/facade/context].

Composition:
Use the [pattern name] pattern.
[pattern composition guidance]

Variation directives:
Generate three concept directions.
A: name-first layout.
B: category-recognition layout.
C: high-contrast readability layout.

Production notes:
Treat this as concept direction only, not final sign fabrication.
Prioritize typography hierarchy, composition, visual impact, and commercial readability.
Keep the concept easy for a designer to refine in Illustrator.
```
