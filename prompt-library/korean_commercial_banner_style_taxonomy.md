# Korean Commercial Banner Style Taxonomy

# Purpose

Reusable style categories for Korean promotional banner and sign design prompts.

Use this taxonomy inside Prompt Builder style selectors and image2 prompt layers.
These categories are for concept direction, not final artwork generation.

# Style Category Format

Each category includes:

- use case
- visual intent
- typography direction
- color direction
- composition guidance
- prompt phrase
- avoid

--------------------------------------------------
# 1. Bold Retail Sale
--------------------------------------------------

## Use Case

Discount sales, clearance events, limited-time offers, mart/store promotions.

## Visual Intent

Immediate price and offer recognition.

## Typography Direction

Heavy bold Korean headline type, large numerals, compact supporting text.

## Color Direction

High-contrast red, yellow, white, black, or charcoal.

## Composition Guidance

Make the discount or main offer the largest element.
Use badges, blocks, or price zones only if they improve scanning speed.

## Prompt Phrase

```text
Use a bold Korean retail sale style with oversized headline typography, large price emphasis, strong red/yellow contrast, and fast street-level readability.
```

## Avoid

- decorative detail
- weak contrast
- small legal-style copy
- premium minimal spacing that reduces urgency

--------------------------------------------------
# 2. Clean Premium Local
--------------------------------------------------

## Use Case

Clinics, salons, studios, cafes, boutiques, local service businesses.

## Visual Intent

Trust, cleanliness, and refined professionalism.

## Typography Direction

Clean sans-serif Korean type, moderate weight, generous spacing.

## Color Direction

Warm white, black, muted beige, muted green, navy, or soft metallic accents.

## Composition Guidance

Use a name-first or message-first layout with restrained graphic elements.
Keep visual hierarchy calm but clear.

## Prompt Phrase

```text
Use a clean premium Korean local-business style with restrained typography, generous spacing, muted colors, and a trustworthy professional tone.
```

## Avoid

- aggressive discount graphics
- crowded promotional copy
- loud gradients
- excessive illustration

--------------------------------------------------
# 3. Friendly Neighborhood Promo
--------------------------------------------------

## Use Case

Local cafes, bakeries, restaurants, small shops, community events.

## Visual Intent

Approachable, warm, and easy to understand.

## Typography Direction

Rounded or friendly Korean headline type with clear supporting text.

## Color Direction

Warm yellow, cream, green, coral, brown-neutral, or soft blue accents.

## Composition Guidance

Use simple icons or product cues around the main message.
Keep the headline readable and avoid making the design too cute for commercial use.

## Prompt Phrase

```text
Use a friendly Korean neighborhood promotion style with warm colors, approachable headline typography, simple product cues, and clear local-shop readability.
```

## Avoid

- childish visuals
- overly soft contrast
- too many small icons
- cluttered handwritten effects

--------------------------------------------------
# 4. High-Impact Event
--------------------------------------------------

## Use Case

Festivals, pop-ups, concerts, sports events, seasonal markets.

## Visual Intent

Energy, movement, and quick event recognition.

## Typography Direction

Bold display Korean headline type with strong rhythm and clear date hierarchy.

## Color Direction

Vivid contrast using two strong colors plus one neutral.

## Composition Guidance

Make event name and date easy to scan.
Use diagonal, radial, or layered composition only if readability remains strong.

## Prompt Phrase

```text
Use a high-impact Korean event banner style with energetic display typography, vivid contrast, strong date hierarchy, and dynamic but readable composition.
```

## Avoid

- poster-like complexity
- hidden dates
- low-contrast atmosphere
- decorative motion that weakens the headline

--------------------------------------------------
# 5. Food Promotion
--------------------------------------------------

## Use Case

Restaurant specials, delivery promos, menu launches, cafe seasonal drinks.

## Visual Intent

Appetite appeal plus immediate offer clarity.

## Typography Direction

Bold Korean menu headline, large product name, readable price or benefit.

## Color Direction

Warm reds, oranges, yellows, deep greens, cream, and dark contrast.

## Composition Guidance

Let food or drink imagery support the headline.
Do not let imagery overpower price, menu name, or promo message.

## Prompt Phrase

```text
Use a Korean food promotion style with appetizing warm colors, bold menu typography, clear price or benefit hierarchy, and product-led commercial readability.
```

## Avoid

- overly lifestyle-focused imagery
- weak menu readability
- excessive garnish graphics
- unclear offer structure

--------------------------------------------------
# 6. Urgent Limited-Time
--------------------------------------------------

## Use Case

Flash sales, closing sales, last chance offers, deadline-driven promotions.

## Visual Intent

Urgency without visual chaos.

## Typography Direction

Compressed bold Korean type, large deadline text, strong callout labels.

## Color Direction

Red, black, white, yellow, electric blue accents when appropriate.

## Composition Guidance

Use a deadline-first or offer-first layout.
Keep only one dominant urgency cue.

## Prompt Phrase

```text
Use an urgent Korean limited-time promotion style with compressed bold typography, deadline-first hierarchy, sharp contrast, and direct action-oriented readability.
```

## Avoid

- multiple competing badges
- panic-like clutter
- tiny date text
- unclear main offer

--------------------------------------------------
# 7. Informational Service
--------------------------------------------------

## Use Case

Academies, hospitals, repair services, real estate, B2B local services.

## Visual Intent

Clarity, credibility, and service comprehension.

## Typography Direction

Structured Korean type hierarchy with clear headline, benefit, and contact zones.

## Color Direction

Blue, green, white, charcoal, muted orange or red for emphasis.

## Composition Guidance

Separate service category, key benefit, and contact/action information.
Use icons only when they reduce reading time.

## Prompt Phrase

```text
Use a Korean informational service banner style with structured typography, clear benefit hierarchy, credible colors, and practical contact readability.
```

## Avoid

- vague brand-only layouts
- too much paragraph text
- decorative icons
- emotional imagery that obscures service clarity

--------------------------------------------------
# 8. Youth Trend Promo
--------------------------------------------------

## Use Case

Fashion, beauty, pop-up shops, youth events, trendy cafes, lifestyle promos.

## Visual Intent

Fresh, trendy, and social-media aware while still readable as a commercial banner.

## Typography Direction

Bold contemporary Korean display type with selective English accent text.

## Color Direction

Bright accent colors, black/white contrast, neon accents, or playful pastel contrast.

## Composition Guidance

Use asymmetry, stickers, cutout-like elements, or layered type sparingly.
Keep the main promotion readable at first glance.

## Prompt Phrase

```text
Use a trendy Korean youth promotion style with contemporary display typography, bright accent contrast, selective English text accents, and playful but commercially readable composition.
```

## Avoid

- illegible experimental type
- too many decorative stickers
- style that hides the offer
- generic social media template look
