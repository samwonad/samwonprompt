# AGENTS.md

# SCOPE

This folder owns prompt workflow assets for Design Copilot MVP.

Focus on:

- Prompt Builder patterns
- image2 prompt structures
- reusable prompt templates
- brief-to-prompt transformation examples
- A/B/C concept direction prompts

Do not focus on:

- final artwork generation
- large prompt frameworks
- backend orchestration
- speculative automation

# WORKING RULES

Before adding a prompt, define:

1. the designer input
2. the intended concept output
3. how it saves concept-start time

Prefer:

- short reusable templates
- clear variables
- mock examples
- designer-readable language

Avoid:

- vague style words without design meaning
- prompts that imply final production artwork
- excessive prompt chains

# ACCEPTANCE

A prompt asset is useful only if it helps a designer reach a usable starting concept faster.
