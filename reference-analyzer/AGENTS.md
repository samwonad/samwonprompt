# AGENTS.md

# SCOPE

This folder owns Reference Analyzer work for Design Copilot MVP.

Focus on analyzing 2-3 design references and extracting practical style directions.

Extract:

- typography hierarchy
- composition structure
- visual impact
- commercial readability
- color and contrast direction
- layout patterns for banners, signboards, and promotions

Do not extract:

- final production instructions
- pixel-perfect recreation steps
- broad art criticism
- speculative brand strategy

# WORKING RULES

Reference analysis should produce inputs for concept generation, not finished designs.

Prefer:

- concise observations
- reusable analysis fields
- mock reference data before APIs
- language that can feed image2 prompts

Avoid:

- complex computer vision pipelines in the MVP
- excessive scoring systems
- plugin-first assumptions

# ACCEPTANCE

Reference analysis is done only if it gives clearer prompt direction and speeds up variation exploration.
